The genomics-proteomics toolkit

About GTO2

GTO2 is an improved version of GTO, which is a toolkit created for genomics and proteomics, namely for FASTQ, FASTA and SEQ formats. The toolkit is for Unix-based systems, built for ultra-fast computations. GTO2 supports pipes for easy integration with the sub-programs belonging to GTO as well as external tools. GTO2 works as LEGOs, since it allows the construction of multiple pipelines with many combinations. GTO2 is also available in the Conda package manager for easy use in the different operating systems.
GTO2 includes tools for information display, randomisation, edition, conversion, extraction, search, calculation, compression, simulation and visualisation. GTO2 is prepared to deal with very large datasets, typically in the scale of Gigabytes or Terabytes (but not limited). The complete toolkit is an optimised command-line version, using the prefix "gto2_" followed by the suffix with the respective name of the program. GTO2 is implemented in C language and it is available, under the MIT license.

Installation

GTO2 can be installed using the Cobilab channel from Conda:

conda install -c cobilab gto2 --yes

Or through the GitHub repository:

git clone https://github.com/cobilab/gto.git
cd gto/src/
make
